# PLUX API Library Files

This directory should contain the PLUX API library files required for building the application.

## Required Library Files

- For Windows 32-bit: `plux.lib`
- For Windows 64-bit: `plux64.lib`
- For macOS: `libplux.a`
- For Linux: `libplux.a`

## How to Obtain Library Files

1. Download the PLUX API from this link: https://downloads.plux.info/apis/PLUX-API-Cpp.zip
2. Extract the archive
3. Copy the appropriate library files for your platform into this directory

If you need the API compiled for other architectures, contact support@plux.info
